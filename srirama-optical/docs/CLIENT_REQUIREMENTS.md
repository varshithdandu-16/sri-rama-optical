# Project Objective

Develop a modern, premium, mobile-first optical showroom website that helps customers:

- Explore products
- Explore services
- Learn about eye testing
- Build trust in the business
- Contact the store directly
- Visit the physical store

This website is intended to act as a digital showroom and customer inquiry platform.

# Customer Communication Requirements

## Direct Calling

Users should be able to directly call the store.

When a user clicks:

- Phone Number
- Call Now Button

The website should open the phone dialer automatically.

Example:

Call Now
↓
Mobile Dial Pad Opens
↓
Customer Calls Store

---

## Direct Navigation

Users should be able to directly navigate to the store.

When a user clicks:

Get Directions

The website should open Google Maps directly.

Example:

Get Directions
↓
Google Maps Opens
↓
Navigation Starts
